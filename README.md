# volto.buildout

Simple buildout for Volto projects

## Usage

Install virtualenv

```bash
python3 -m virtualenv .
```

Activate virtualenv
```bash
. bin/activate
```

Install requirements
```bash
pip install -r requirements.txt
```

Run buildout
```bash
bin/buildout -N
```

## Running Plone

Run Plone instance
```bash
bin/instance fg
```
